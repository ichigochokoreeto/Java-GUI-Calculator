# Java-GUI-Calculator
A simple Java desktop calculator built with Swing as a beginner

# Features
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Clear button (AC)
6. Equals button (=)

# How to Run
1. Open IDE (Netbeans)
2. Create new project named "CalculatorGUI"
3. Copy and paste the code
4. Compile and run the code
